Connect Google Drive

Paste your Google OAuth Client ID and the ID of your top-level podcast folder. Your files stay private.

One-Time Setup (≈ 20 min)

1

Go to console.cloud.google.com → Create a new project → Enable Google Drive API

2

Go to APIs & Services → Credentials → Create OAuth 2.0 Client ID → Application type: Web application

3

Under Authorized JavaScript origins, add the URL where you host this file (e.g. your GitHub Pages URL)

4

Copy the Client ID and paste it above

5

Open your podcast folder in Drive → copy the ID from the URL (the part after /folders/)
